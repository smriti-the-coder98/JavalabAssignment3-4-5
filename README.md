# Student Management System (Java)

This is a simple Java program that uses:

Custom Exception

Multithreading (Loader animation)

HashMap to store student records

Input validation (name, email, marks, roll number)

The program lets you:

Add a student

Display a student using roll number

Handle errors if the student is not found

Add new student

View all students

Search student by name

Delete student by name

Sort students by marks

Save all records to file

Load records from file

RandomAccessFile demo (reads first line)
